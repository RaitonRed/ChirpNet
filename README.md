# ChirpNet - پروژه شبکه اجتماعی شبیه توییتر

**ChirpNet** یک پلتفرم شبکه اجتماعی شبیه توییتر است که با استفاده از Django برای بک‌اند و React برای فرانت‌اند ساخته شده است. کاربران می‌توانند ثبت‌نام کنند، توییت‌های کوتاه (حداکثر ۲۸۰ کاراکتر) ارسال کنند، توییت‌ها را لایک کنند و فید توییت‌ها را مشاهده کنند.

### ویژگی‌ها:
- **احراز هویت کاربران**: کاربران می‌توانند ثبت‌نام کنند، وارد حساب کاربری خود شوند و پروفایل شخصی خود را مشاهده کنند.
- **ارسال توییت**: کاربران می‌توانند توییت‌های کوتاه ارسال کنند و آن‌ها را در فید مشاهده کنند.
- **لایک کردن توییت‌ها**: کاربران می‌توانند توییت‌های دیگران را لایک کنند و آن‌ها را در پروفایل خود مشاهده کنند.
- **پروفایل کاربر**: هر کاربر صفحه پروفایل خود را دارد که شامل توییت‌های ارسال‌شده و لایک‌های انجام‌شده است.
- **API RESTful**: بک‌اند با استفاده از Django REST Framework ساخته شده است تا ارتباط میان فرانت‌اند و بک‌اند برقرار باشد.

---

### نقشه راه پروژه:

1. **مرحله اول: راه‌اندازی بک‌اند**
    - ایجاد پروژه Django و تنظیمات اولیه
    - ساخت مدل‌های دیتابیس برای توییت‌ها، لایک‌ها و کاربران
    - ایجاد API‌های مربوط به توییت‌ها (ارسال، لایک، مشاهده فید)
    - پیاده‌سازی احراز هویت JWT برای ورود و ثبت‌نام کاربران
    - نوشتن سریالایزر‌ها و ویو‌ها برای API
    - ایجاد مسیرهای (URLs) مورد نیاز برای API

2. **مرحله دوم: راه‌اندازی فرانت‌اند**
    - ایجاد پروژه React
    - طراحی صفحات اصلی: ورود (Login)، فید توییت‌ها (Home)، پروفایل کاربری
    - استفاده از axios برای ارتباط با API بک‌اند
    - طراحی فرم ارسال توییت
    - طراحی کامپوننت‌های قابل استفاده مجدد مثل Navbar و TweetCard

3. **مرحله سوم: اتصال بک‌اند به فرانت‌اند**
    - اتصال API‌های Django به صفحات React
    - مدیریت وضعیت (State) برای ذخیره اطلاعات کاربری و توییت‌ها
    - پیاده‌سازی سیستم لایک کردن توییت‌ها در فرانت‌اند

4. **مرحله چهارم: استایل‌دهی و بهینه‌سازی**
    - اضافه کردن استایل‌های مدرن با استفاده از CSS یا Tailwind CSS
    - بهینه‌سازی کدها برای عملکرد بهتر در دستگاه‌های مختلف
    - تست‌کردن سیستم لایک و ارسال توییت در حالت‌های مختلف

5. **مرحله پنجم: دیپلوی پروژه**
    - راه‌اندازی پروژه روی سرور (Heroku یا DigitalOcean)
    - تنظیمات نهایی برای انتشار API و فرانت‌اند
    - تست نهایی و رفع مشکلات احتمالی

### شروع پروژه:

1. **کلون کردن پروژه:**

   برای شروع، این پروژه را کلون کرده و به دایرکتوری پروژه بروید:

   ```bash
   git clone https://github.com/RaitonRed/ChirpNet.git
   cd ChirpNet
    ```
---

2. نصب وابستگی‌های بک‌اند:

    در دایرکتوری backend وارد شوید و وابستگی‌های پروژه را نصب کنید:
    ```bash
    cd backend
    pip install -r requirements.txt
    ```

---

3. راه‌اندازی بک‌اند:

    برای راه‌اندازی سرور بک‌اند Django:
    ```bash
    python manage.py runserver
    ```
---

4. نصب وابستگی‌های فرانت‌اند:

    در دایرکتوری frontend وارد شوید و وابستگی‌های React را نصب کنید:
    ```bash
    cd frontend
    npm install
    ```

---

5. راه‌اندازی فرانت‌اند:

    برای راه‌اندازی سرور React:
    ```bash
    npm start
    ```
